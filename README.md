# hasuta.shop

This is the repository for hasuta.shop project.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/mdsufiyan3/h-ecommarce.git
```

2. Install dependencies:
```bash
npm install
```

## Usage

To run the application:
```bash
npm start
```
